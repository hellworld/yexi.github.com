Android color相关

Android中使用4个数字来表示颜色，分别是alpha,red,green,blue四个颜色值（ARGB）。每个数值取值0-255，因此一个颜色可以用一个整数来表示。红绿蓝三个值代表了颜色的取值，而alpha代表透明度，0表示颜色完全透明，而此时的RGB是什么取值都不重要了，255表示颜色完全不透明。

在Android以下的方式可创建或表示颜色：

  1. 使用Color类的常量:

     Int color = Color.BULE; //创建一个蓝色

  2. 如果知道ARGB的取值，可以使用Color类的静态方法argb创建一个颜色:

     Int color ＝ Color.argb(127, 255, 0, 255); //半透明紫色

  3. 使用XML资源文件来表示颜色，这是一个扩展比较好的方式，便于今后可方便修改颜色值:

     <?xml version="1.0" encoding="utf-8"?>
 <resources>
 <color name="white">#FFFFFF</color><!--白色 -->
 </resources>

    定义一个名为mycolor，在代码中引用:

         Int color = getResources().getColor(R.color.mycolor);
