关于Android欢迎界面和两次Back退出

看到一篇博文不错，正好自己最近也有碰到

点我

SplashActivity：功能是在执行显示欢迎界面一定时间（LOADINGTIME决定） 启动主界面MainActivity代码如下：

 private void StartMainActivity() {
    new Handler().postDelayed(new Runnable() {
        @Override
        public void run() {
            Intent intent = new Intent(SplashActivity.this, MainActivity.class);
            startActivity(intent);
            SplashActivity.this.finish();
        }
    }, LOADING_TIME);
}

//intent在启动之前要设置一下Activity的标志，具体查看Activity任务栈问题 intent.addFlags(Intent.FLAG_ACTIVITY_CLEAR_TOP); 就这样问题轻松解决了

顺便Mark一下，如何在应用程序中“按两次返回键退出程序，并拦截Back键”

private int mBackKeyPressedCount = 1;
@Override
public void onBackPressed() {
    if (mBackKeyPressedCount == 2){
        mRequstQueue.cancelAll(this);
        if (player != null) {
            player.stop();
            player = null;
        }
        finish();
    } else {
        mBackKeyPressedCount++;
        Toast.makeText(this,"再按一次退出程序", Toast.LENGTH_SHORT).show();
    }
}
